OpenRocket is developed and maintained entirely by volunteers in their free time. Every contribution — whether code, documentation, translations, testing, or design examples — directly improves the tool for the entire rocketry community.

## Ways to contribute

<CardGroup cols={2}>
  <Card title="Code" icon="code" href="https://github.com/openrocket/openrocket">
    Fix bugs, implement new features, or improve existing functionality. The source code is on GitHub and contributions are made via pull requests.
  </Card>
  <Card title="Documentation" icon="book">
    Help improve or expand this documentation. If you notice something missing or incorrect, you can contribute directly. See the Developer Guide for instructions.
  </Card>
  <Card title="Translations" icon="language" href="https://crowdin.com/project/openrocket">
    Help translate the OpenRocket interface into other languages via the OpenRocket Crowdin project. Translations are community-maintained and many are incomplete.
  </Card>
  <Card title="Testing" icon="flask" href="https://github.com/openrocket/openrocket/releases">
    Test pre-release versions of OpenRocket and report bugs on the GitHub issue tracker. Pre-release testing helps catch last-minute issues before a stable release.
  </Card>
  <Card title="Example designs" icon="drafting-compass">
    Create well-documented example rocket designs that demonstrate OpenRocket features and can be bundled with the application.
  </Card>
  <Card title="Bug reports" icon="bug" href="https://github.com/openrocket/openrocket/issues">
    Report bugs or request features on the GitHub issue tracker. Detailed, reproducible bug reports are extremely valuable.
  </Card>
</CardGroup>

## Getting started

The best place to start is the [OpenRocket contribute page](https://openrocket.info/contribute.html) and the [CONTRIBUTING.md](https://github.com/openrocket/openrocket/blob/master/CONTRIBUTING.md) file in the repository. These describe the practicalities of submitting code, the development workflow, and the project's conventions.

For questions and discussion with other contributors, join the [OpenRocket Discord server](https://discord.gg/qD2G5v2FAw).
