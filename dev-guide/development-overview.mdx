Welcome to the OpenRocket developer guide. This documentation covers the architecture, codebase, and development workflows for developers interested in contributing to OpenRocket or embedding it in another application.

For technical details about aerodynamic calculations and simulation methods, see the [OpenRocket technical documentation](https://openrocket.info/documentation.html).

## Code structure

OpenRocket is a Java application organized using the **Java Platform Module System (JPMS)** and built with [Gradle](https://gradle.org/). The codebase is split into two modules:

<CardGroup cols={2}>
  <Card title="info.openrocket.core" icon="microchip" href="/dev-guide/codebase-walkthrough">
    The backend library. Contains the rocket component model, simulation engine, file format parsers and writers, and aerodynamic calculators. Has no dependency on any GUI toolkit — it can be embedded in other applications.
  </Card>
  <Card title="info.openrocket.swing" icon="window" href="/dev-guide/codebase-walkthrough">
    The graphical user interface. Built with the Java Swing toolkit. Depends on `info.openrocket.core` and uses it to run simulations and display results.
  </Card>
</CardGroup>
